  #### 步骤：
  1.安装python3.10+（作者使用的是python3.12.0）  
  2.安装mitmproxy  
  3.安装adb命令  
  4.获取手机或模拟器的root权限  
  5.将mitmproxy证书安装到手机或模拟器的根目录中  
  6.清除手机或模拟器的小猿口算的缓存  
  7.启动mitmproxy，并调用该项目脚本  
  8.解决其他问题
